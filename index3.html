Описание об использовании CSS

CSS (Cascading Style Sheets) - Каскадные Таблицы Стилей, формальный язык описания внешнего вида документа, дополнение для html. CSS обладает более глубокими возможностями оформления сайтов, а так же более лёгким и удобным способом. CSS используется для задания цветов, шрифтов, расположения отдельных блоков и других аспектов представления внешнего вида этих веб-страниц.

Основной целью разработки CSS является разделение описания логической структуры веб-страницы от описания внешнего вида этой веб-страницы. Такое разделение может увеличить доступность документа, предоставить большую гибкость и возможность управления его видом, а так же упрощение структуры самого сторения.

Некторые из функций можно заменить в html на css, так как это будет более удобно и просто. Здесь можно задавать выравнимвание относительно каждого блока или же слова, вплоть до пикселя. Уверен что это очень полезно, так как можно быть уверенным в том, что каждый блок будет расположен именно так как нужно.

В меню слева по пунктам расписанны свойства для оформления всех необходимых элементов, каждое из которых, поможет оформить всё так как нужно. Там есть много информации о свойствах и особенностях написания оформления.
